haXe Video is an Open Source Flash Video (FLV) streaming and recording server.

What

With haXe Video you can :

  • stream videos files to any browser that have Flash Player installed
  • record microphone and webcam as FLV files
  • build web conference service with Live streaming

How

You can download the following files :

  • download the latest haXe Video release
  • download the haXe Video sources from SVN
  • follow the haXe Video Tutorial

haXe Video is entirely written in the haXe programming language.
